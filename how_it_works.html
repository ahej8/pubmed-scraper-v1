Scraping Algorithm

  1. The user inputs search parameters (protein target, max results, max author publications).
  2. The scraper generates a PubMed search query based on the input.
  3. It sends requests to PubMed and parses the search results pages.
  4. For each article found:
    • Extracts title, authors, and link
    • Retrieves the full article page to extract the abstract and email addresses
    • Analyzes the abstract to determine potential antibody needs
    • Checks the lead author's publication count
  5. Results are streamed back to the user's browser in real-time.
  6. The data can be viewed in the browser or exported to an Excel file.

The scraper uses libraries like BeautifulSoup for HTML parsing and NLTK for text analysis. It respects PubMed's robots.txt and includes delays between requests to avoid overloading their servers.
